# Nav Quest 3D

A quick [Godot](https://godotengine.org/) project one may use to better understand the complexities of the `3D` viewport navigation.


## Instructions

Detailed project instructions begin here - [Nav Quest 3D](https://codeberg.org/leikskoli/dev-drills/src/branch/main/Booting/6.%20Nav%20Quest%203D/README.md).


### 3D Viewport Navigation
- Orbit: `MMB` + drag
- Pan: `Shift` + `MMB` + drag
- Zoom In: `MMW` + up
- Zoom Out: `MMW` + down
- Focus: `F`
- To swap the orbit & pan hotkeys open the menu: `Editor` > `Editor Settings...` > `General` tab > `Editors` > `3D` > `Navigation`
  - Orbit Modifier: `Shift`
  - Pan Modifier: `None`
